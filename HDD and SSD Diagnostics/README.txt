==================================================
HARD DISK SENTINEL (PORTABLE) GUIDE
==================================================

Hard Disk Sentinel (HDSentinel) is a professional HDD/SSD monitoring and analysis software. 
It is designed to check, diagnose, and repair hard disk drive and SSD problems, reporting and displaying drive health, performance degradations, and temperature to prevent data loss.

--------------------------------------------------
HOW TO USE (PORTABLE VERSION):
--------------------------------------------------

1. LAUNCH THE PROGRAM:
   - Double-click "HDSentinel.exe" (or right-click and select "Run as administrator").
   - Since this is a portable version, it runs directly without installation.

2. RUN IN SYSTEM TRAY (OPTIONAL):
   - Double-click "HDSentinelTray.exe" to run the software minimized in your Windows System Tray (taskbar).
   - This will display real-time drive temperatures in your taskbar.

--------------------------------------------------
KEY FEATURES & DIAGNOSTIC SECTIONS:
--------------------------------------------------

- LEFT SIDEBAR:
  Displays a list of all connected drives (HDD, SSD, NVMe, and USB external drives) with their overall Health (%), Performance (%), and Temperature.

- OVERVIEW TAB (Right Pane):
  Provides a textual diagnosis and reports any issues found (e.g. bad sectors, interface communication errors). If Health is at 100%, the drive is in perfect condition.

- TEMPERATURE TAB:
  Tracks current, average, and maximum temperatures. Keeping drives under 45-50°C is recommended for longevity.

- S.M.A.R.T. TAB:
  Displays self-monitoring parameters (like Power-on time, Reallocated Sector Count, etc.) for advanced users to inspect drive health trends.

- DISK TEST TOOLS (Top Menu -> Disk):
  Allows you to perform hardware tests:
  - "Short Self-Test" / "Extended Self-Test": Diagnostic routines built into the drive.
  - "Surface Test": Verifies the readability of all sectors (useful for finding/fixing bad sectors).
