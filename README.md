# Useful Files & Utilities

This repository serves as a centralized, curated collection of reusable utilities, helper scripts, templates, configuration files, and software tools. Maintaining these resources in a single, version-controlled location ensures quick access, prevents redundant searches, and simplifies workspace configuration or troubleshooting across different machines.

## Repository Contents

*   [cleaner.bat](./Cleaner%20for%20PC/cleaner.bat) – An interactive maintenance utility that clears system temp files, flushes network/DNS caches, cleans browser caches, runs Disk Cleanup, executes system health checks (SFC/DISM), and optimizes storage drives.
*   [Excel Dropdown Printer](./Excel%20Dropdown%20Printer) – A folder containing an Excel VBA macro (`README.txt`) to automatically loop through a dropdown list and print a sheet/template for each entry.
*   [Activation](./Activation) – A folder containing system activation helper tools, including a command (`activation.txt`) and a `README.txt` usage guide.
*   [Resetter for Epson Printers](./Resetter%20for%20Epson%20Printers) – A folder containing Epson printer maintenance tools, including a ZIP archive (`Epson Resetter.zip`) with a password-protected resetter program, and a `README.txt` guide.
*   [HDD and SSD Diagnostics](./HDD%20and%20SSD%20Diagnostics) – A folder containing drive diagnostic and monitoring tools, including a portable Hard Disk Sentinel ZIP archive (`sentinel.zip`) and a `README.txt` usage guide.
